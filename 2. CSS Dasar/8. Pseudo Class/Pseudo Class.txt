 >> PENERAPAN PSEUDO CLASS <<


 >> Pseudo Class = Kelas semu yg dimiliki elemen HTML, yg membuat kita dpt 
		   mendefinisikan style pada "keadaan tertentu" dari elemen 
		   tersebut

 >> Kelas semu = Akan aktif ketika keadaan tertentu

 >> Tipe pseudo class : 

    => Berhubungan dgn link : 

       1. :link = Style default pada link(a yg punya href)

       2. :hover = Style ketika kursor mouse berada di atas link/elemen

       3. :active = Style ketika link di-klik (keadaan aktif)

       4. :visited = Style ketika link sudah pernah dikunjungi sebelumnya(pada 
		     browser yg sama)

    => Berhubungan dgn posisi elemen(1) : 

       1. :first-child = Memilih elemen pertama dari parent(elemen 
			 pembungkusnya)

       2. :last-child = Memilih elemen terakhir dari sebuah paren(elemen 
			pembungkusnya)

       3. :nth-child(n) = Memilih elemen ke-n dari sebuah parent(elemen 
			  pembungkusnya)

	  => n  =  urutan(1, 2, ...), pola((2n), (3n+2), (4n-1)), atau 
		   ganjil(odd) dan genap(even)

    => Berhubungan dgn posisi elemen(2) : 

       1. :first-of-type = Memilih elemen pertama dari sebuah jenis/tipe tag

       2. :last-of-type = Memilih elemen terakhir dari sebuah jenis/tipe tag

 >> Penerapan : 

    => Buat folder yg di dlmnya ada file HTML dan CSS, di dlm file HTML ada 
       tag a yg di dlmnya diisi "Hello World", lalu ada tag ul dan di dlmnya 
       ada tag li sebanyak 10, dan di dlmnya lagi ada tag a yg berisi tulisan 
       link 1-10. Ada tag paragraph sebanyak 2 yg menyimpan lorem. Hubungkan 
       file html dan css menggunakan tag link

    1. Contoh :link : 

		.hello:link{
			color: orange;
		}

       => :link akan bekerja ketika elemennya tag a

    2. Contoh :hover : 

		.hello:hover{
			font-family: arial;
			font-size: 20px;
		}

       => Hasilnya tidak ada perubahan, karena hover akan dipanggil ketika 
	  kursor mengarah ke isi konten. Jika kursor mengarah ke konten, maka 
	  font akan berubah jadi arial dan ukurannya akan besar 20 pixel

    3. Contoh :active : 

		.hello:active{
			font-style: italic;
		}

       => Ketika kursor klik konten pada tag a yg memiliki class hello, akan 
	  merubah gaya font menjadi huruf miring

    4. Contoh :visited : 

		.hello:visited{
			color: red;
		}

       => Hasilnya ketika tag a dgn class hello sudah pernah dikunjungi oleh 
	  browser, maka warna akan berubah menjadi merah

    5. Contoh :first-child : (tag li memiliki anak 1 a, tag ul memiliki anak 
			     10 li)

		li:first-child a{
			color: green;
		}

       => Artinya, tolong carikan tag li tapi yg merupakan anak pertama, lalu 
	  a di dlmnya, ganti warnanya menjadi hijau

    6. Contoh :last-child :

		li:last-child a{
			color: green;
		}

       => Artinya, tolong carikan tag li tapi yg merupakan anak terakhir, lalu 
	  a di dlmnya, ganti warnanya menjadi hijau

    7. Contoh :nth-child(n) :

		li:nth-child(3) a{
			color: green;
		}

       => Artinya, tolong carikan tag li tapi yg merupakan anak ke-3, lalu a 
	  di dlmnya, ganti warnanya menjadi hijau

    8. Contoh :nth-child(even) :

		li:nth-child(even) a{
			color: green;
		}

       => Artinya, tolong carikan tag li tapi yg merupakan anak genap, lalu a 
	  di dlmnya, ganti warnanya menjadi hijau

    9. Contoh :nth-child(odd) :

		li:nth-child(odd) a{
			color: green;
		}

       => Artinya, tolong carikan tag li tapi yg merupakan anak ganjil, lalu a 
	  di dlmnya, ganti warnanya menjadi hijau

    >> Ketika kita gunakan nth-child(n) dgn hover, maka kita bisa terapkan 
       ketika membuat table yg setiap kotaknya warna nya belang saat di hover

    >> Ketika kita ingin memilih elemen p bagian awal, dan mengganti warnanya, 
       maka :first-child tidak bisa diterapkan karena tag p tidak berada di 
       parent manapun, solusinya gunakan :first-of type

    10. Contoh :first-of-type : 

		p:first-of-type{
			color: red;
		}

       => Cari elemen p yg pertama, ubah warnanya menjadi merah

    10. Contoh :last-of-type : 

		p:last-of-type{
			color: red;
		}

       => Cari elemen p yg terakhir, ubah warnanya menjadi merah
